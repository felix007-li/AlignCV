## 🚀 使用指南

### 1. 启动开发服务器

```bash
cd frontend
npm run start
```

### 2. 选择彩色侧边栏模版

点击 **"Templates"** 按钮，选择以下任意模版：

**推荐顺序**:
1. ⭐ **Modern** - 深棕红色（最像CVWizard！）
2. **Elegant** - 深蓝灰色 + 灰白背景
3. **AltaCV** - 鲜绿色右侧栏
4. **Deedy** - 深橙红色右侧栏
5. **Awesome CV** - 专业蓝色
6. **Timeline** - 现代青绿色
7. **Friggeri** - 低调深灰
8. **Twenty Seconds** - 热情红色

### 3. 选择大字体单栏模版

如果不需要侧边栏：
1. **Even** - 最大字体（14/19px）
2. **ModernCV** - 最大字体 + serif
3. **Dev ATS** - ATS优化
4. **Compact** - 紧凑但清晰

### 4. 查看效果

**彩色侧边栏模版**:
- 左侧/右侧：深色背景 + 白色文字
- Contact, Skills, Languages显示在侧边栏
- 大字体，清晰易读

**单栏模版**:
- 全宽布局
- 大字体，专业
- 彩色bar标题
