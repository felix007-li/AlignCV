# 商业化与增长

## 定价与包装
- **学生一次性**：$1.99（验证/声明 + 限制每次 1 份导出）
- **公众一次性**：$2.99（单次导出）
- **月订阅**：$7.99（不限导出/模板主题解锁）
- 多币种适配：USD/CAD/EUR/MXN/ARS/CLP/BRL（Stripe 自动识别价格表）

## 漏斗与目标（首季）
- 访问 → 导入 → 生成建议 → 查看/应用 → 导出 → 付费
- 目标基线：
  - 导入转化率 ≥ 45%
  - 应用 ≥ 2 条建议/用户
  - 学生付费转化 ≥ 6%（SEO渠道），公众付费 ≥ 3%
  - SEO 自然流量 20k/月，模板页跳出 < 55%

## 获客
- **SEO**：城市×行业×资历 长尾（3/5 段路由 + SSR）
- **付费流量**：Google Ads（“resume + role/city”），Meta（短视频对比）
- **校园渠道**：学校合作、学生组织码；导师/教务 Dashboard（v3.0）
- **联盟/网红**：产品测评 & “1 分钟改简历”挑战

## A/B 策略
- 付费墙 A/B/C（i18n 文案差异）→ 指标：`paywall_impression → cta_click → checkout`
- 建议面板：默认候选数 3 vs 5；按钮文案/位置对转化影响

## 合作线索
- 招聘平台/ATS：导出模板与品牌联合页面
- Bootcamp/大学：批量授权 + 品牌页

## 风险与对策
- 版权/抄袭：模板以结构/提示为主，示例文本保持通用；提供原创声明与来源标记。
- 生成质量波动：失败→启发式回退；前端提示可二次编辑。
